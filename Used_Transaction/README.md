교내 중고거래 O2O APP
=====================

Description
------------
학교 내에서 학생들이 중고 거래를 편리하게 할 수 있는 어플리케이션입니다.

구매자는 어디서든 앱을 통해 원하는 물품을 선택하여 판매자와 컨택하고

실제 거래는 교내에서 오프라인으로 안전하게 할 수 있습니다.
